## Description (Markdown)

**Genable** brings the power of Google's Gemini AI directly into Figma, allowing you to generate production-ready UI components simply by describing them.

✨ **Key Features:**
*   **Chat-to-Design:** Just describe what you need (e.g., "A modern pricing card with 3 tiers") and watch it appear.
*   **Editable Layers:** Generates real Auto Layout frames, text, and vector shapes—not images.
*   **Context Aware:** Select an existing layer, and Genable will try to match its width, fonts, and colors.
*   **Secure:** Your API key is stored locally on your device.

🛠 **How to Use:**
1.  Run the plugin.
2.  Enter your Gemini API Key (get one for free at `aistudio.google.com`).
3.  Type a prompt or select a template.
4.  Click **Generate**.
