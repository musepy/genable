## Full Description

Type what you need. Get a complete Figma design.

Genable is an AI design generator that creates fully editable UI components from natural language descriptions. Dashboards, forms, landing pages, mobile screens, pricing tables — just describe it and get production-ready Figma layers you can immediately customize.

### What you get

- **Real Figma layers** — not flattened images. Move, resize, and restyle every element
- **Auto-layout built in** — responsive designs that resize properly
- **Conversational refinement** — say "make the CTA bigger" or "add dark mode" to iterate
- **Multi-model support** — works with Gemini, Claude, and other LLMs. Bring your own API key
- **Free to use** — no subscription, no credits, no limits

### Best for

- Rapid UI prototyping and exploration
- Dashboard and data visualization mockups
- Landing page and marketing page design
- Form and settings page generation
- Mobile app screen design
- Design system-aligned component creation

### How it works

1. Open Genable from the Figma plugin menu
2. Describe the UI you need in plain English
3. Genable generates editable Figma components with proper auto-layout, spacing, and styling
4. Refine with follow-up messages — "change the color scheme" or "make it more compact"

### Bring your own API key

Genable is completely free. You provide your own LLM API key (Gemini, Claude, etc.) and generate unlimited designs with no usage limits.
