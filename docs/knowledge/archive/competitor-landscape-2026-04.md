## 三、共同的文案策略

### 1. 标题关键词堆叠（SEO 优化）
几乎所有插件都在 Figma Community 标题中堆叠高搜索量关键词：
- **高频词**: AI, UI Generator, Design, Wireframe, Prompt, Text to, Image to, Mobile App, Redesign
- **典型模式**: `[品牌]: [功能1], [功能2], [功能3] | Figma`
- **最极端案例**: DesignFast 标题含 9 个关键词 —— "AI UI Generator (UI Pilot, Prompt to UI, Image to UI, Mobile App, AI Redesign)"
- UX Pilot 频繁更换标题优化 SEO（搜索结果中出现 3 种不同标题变体）

### 2. 数字化承诺
- "3 秒生成"（Codia）
- "10x faster"（Superun）
- "70% 减少迭代时间"（UX Pilot、Codia）
- "75% 减少设计迭代时间"（UX Pilot）
- "3x 更快原型制作"（UX Pilot）
- "80K+ Figma 组件已生成"（UX Pilot）

### 3. "可编辑"反复强调
所有插件都反复强调输出是"fully editable Figma layers/components"，而非静态图片。这是核心价值主张——区别于 AI 图片生成工具（如 Midjourney）。

### 4. 多输入方式展示
几乎所有插件都支持并宣传多种输入：
- Text prompt（核心）
- Screenshot/Image → Design
- URL → Design
- Sketch → Design
- PRD → Design（Ugic、Motiff）

### 5. "No coding required" / "No credit card"
降低试用门槛的标准话术。

### 6. 社会证明
- "Trusted by X+ designers/users"
- "From 181 countries"（Codia）
- "50,000+ / 300,000+ / 600,000+"（按规模递增）

## 四、定价模式分布

### 模式统计

| 定价模式 | 插件数量 | 代表 |
|----------|----------|------|
| **Freemium（免费 + 付费升级）** | 8 | UX Pilot, Codia, UXMagic, ai.to.design, DesignFast, prompt.to.design, Text to Design, Motiff (计划) |
| **仅付费** | 1 | Ugic AI |
| **完全免费** | 1 | Claude Design Assistant |
| **平台内置** | 1 | Figma Make |

### 定价区间

| 价格带 | 月价范围 | 插件 |
|--------|----------|------|
| 低价位 | EUR 12 ~ $14/月 | ai.to.design, UX Pilot Standard |
| 中价位 | $19 ~ $29/月 | UX Pilot Pro, UXMagic Standard, Codia Starter, Motiff Professional |
| 高价位 | $31 ~ $50/月 | UX Pilot Teams, UXMagic Pro, Motiff Organization |

### 共同定价特征
1. **额度制为主流** — 几乎所有付费计划按"屏幕数"或"额度/credits"计费
2. **额度不滚存** — 月度额度月底清零（UX Pilot、UXMagic 明确）
3. **免费计划极度限制** — 5~30 次生成，仅够试用（不够完成一个项目）
4. **年付折扣 25-40%** — 标准 SaaS 策略
5. **个人 vs 团队定价分离** — Teams 计划按 seat 计费，通常 +40%~100%

### BYOK 模式（ai.to.design 独有）
ai.to.design 是唯一采用 BYOK（Bring Your Own Key）模式的插件。用户自带 API Key，插件只收取平台费（EUR 12/月 无限渲染）。这意味着：
- 边际成本由用户承担（API 调用费）
- 插件方不承担 LLM 推理成本
- 用户可选择任意模型（成本/质量自由权衡）

## 五、竞争格局洞察

### 1. 头部集中度低
没有绝对霸主。UX Pilot 用户量最大（600K），但远未垄断。市场仍在早期增长阶段。

### 2. Figma Make 是最大变量
Figma 官方的 Make 功能是所有第三方插件的最大威胁——零额外成本、原生集成、无需安装。目前 Make 偏向"原型生成"而非"静态 UI 设计"，但边界在模糊。

### 3. 功能同质化严重
核心功能（text → UI、image → UI、responsive output、editable layers）几乎完全重叠。差异化主要体现在：
- **附加功能**：热力图（UX Pilot）、代码导出（UXMagic）、组件库集成（Ugic）
- **定价策略**：BYOK（ai.to.design）vs 托管式
- **生态定位**：独立平台（Motiff 原来）vs 纯 Figma 插件

### 4. 中国团队占比高
Motiff（中国团队）、Ugic（中国产品）、Codia（多国但有中国市场）——中国团队在这个赛道有显著存在。

### 5. "质量"几乎无人提及
所有插件都在强调"速度"和"易用性"，但几乎没有插件在文案中讨论生成设计的质量（布局合理性、属性完整性、设计规范遵循度）。这可能是差异化机会。

### 6. Google Stitch 的潜在冲击
Galileo AI 被 Google 收购后转型为 Google Stitch。如果 Google 将其深度整合到 Google 生态（与 Gemini、Angular、Flutter 联动），可能成为重量级竞争者。目前仍为免费 beta。
