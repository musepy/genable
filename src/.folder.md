# /src

> Core plugin source - UI Thread (Preact) + Sandbox Thread (Figma) + Agent Engine.
> 双线程架构的核心实现，所有业务逻辑都在这里。

---

## 核心入口 (Entry Points)

| 文件 | 线程 | 功能 |
|------|------|------|
| `main.ts` | Sandbox (Figma) | IPC 消息路由器，DSL → Figma 节点渲染 |
| `ui.tsx` | UI (Preact) | UI 线程入口，界面 + Capture 编排 |
| `types.ts` | 共享 | TypeScript 类型定义，IPC 消息协议 |

---

## 子目录 (Directories)

| 目录 | 功能 |
|------|------|
| `engine/` | Agent 引擎核心：LLM client、agent runtime、tools、pipeline、figma-adapter、sync |
| `features/chat/` | Chat 功能：ChatFeature 组件、useChat hook、logDigest |
| `ui/` | UI 组件层：design-system tokens、组件、布局、utils |
| `generated/` | 构建时生成的知识库 JSON（knowledgeHub 数据源） |
| `config/` | 配置：errorPatterns（错误分类） |
| `constants/` | 常量定义：figma-api、featureFlags、layoutRules 等 |
| `hooks/` | Preact hooks：usePluginData、useModelSettings |
| `ipc/` | IPC handler 层：toolCallHandler、settingsHandler、renderHelper |
| `knowledge/` | 组件知识库：ComponentSchema 类型 |
| `prompts/` | LLM prompt 模板：CORE.md、DESIGN_RULES.md、EXAMPLES.md |
| `schema/` | DSL Schema：layerSchema（Valibot 验证 + 自动修复） |
| `services/` | 服务层：ModelService（模型列表获取） |
| `shared/` | 跨模块共享协议：agentRuntimeEvents、events |
| `types/` | 类型定义：chat、errors、designSystem |
| `utils/` | 工具函数：throttle、colorUtils、jsonStreamParser 等 |

---

## 数据流 (Data Flow)

```
User Prompt (ui.tsx → ChatFeature)
       ↓
useChat hook → AgentOrchestrator
       ↓
Agent Runtime (engine/agent/) ←→ Tool Calls (IPC Bridge)
       ↓ Function Calling
LLM Client (engine/llm-client/) + Knowledge Hub
       ↓ DSL Actions
Action Executor (engine/actions/)
       ↓ IPC: TOOL_RESULT
main.ts → renderHelper → Figma SceneNodes
```
