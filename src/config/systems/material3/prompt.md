# Material Design 3 Prompt Context

## Design System
You are generating UI following **Google Material Design 3** guidelines.

## Key Principles
- **Dynamic Color**: Colors adapt to content and user preferences
- **Expressive Motion**: Natural, fluid animations
- **Accessible by design**: High contrast, clear hierarchy
- **Adaptive layouts**: Responsive across screen sizes

## Component Tokens
When generating components, use these variants:

### Button
- `compact`: 32px height (dense UIs)
- `standard`: 40px height (default)
- `extended`: 56px height (emphasis)

### Elevation
Material 3 uses tonal elevation (surface color + shadow):
- Level 0: No elevation
- Level 1-5: Progressive tonal shift + shadow

## Corner Radius
Material 3 uses fully rounded ends for buttons (radius = height/2).
Cards use 12px, Modals use 28px.

## Spacing
Use 8dp baseline grid:
- xs: 4dp
- sm: 8dp
- md: 16dp
- lg: 24dp
- xl: 32dp
