# Bold Editorial — Style Guide

tags: light-mode, bold, editorial, serif, high-contrast, bold-typography, sharp, geometric

## Description
杂志风格的大胆排版方案，用超大 serif 标题和高对比度黑白基底制造视觉冲击。适合内容平台、博客、新闻阅读、作品集等以文字和图片为核心的产品。

## Color System
| Token | Value | Usage |
|-------|-------|-------|
| bg-page | #FFFFFF | 页面背景 |
| bg-card | #F5F5F5 | 卡片/容器背景 |
| bg-surface | #EBEBEB | 抬升表面（hover、active 状态底） |
| bg-accent | #0A0A0A | 强调色背景（按钮、选中态） |
| text-primary | #0A0A0A | 主文字 |
| text-secondary | #525252 | 次要文字 |
| text-muted | #A3A3A3 | 弱化文字（占位符、禁用态） |
| text-on-accent | #FFFFFF | 强调背景上的文字 |
| border | #D4D4D4 | 边框、分割线 |
| accent | #0A0A0A | 主强调色 |
| accent-hover | #262626 | 强调色 hover |
| success | #15803D | 成功 |
| error | #B91C1C | 错误 |
| warning | #A16207 | 警告 |
