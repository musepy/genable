# Candy Pastel — Style Guide

## Description
糖果色柔和方案，薰衣草紫为主调配合粉色系渐变感。大圆角与 pill 造型传递可爱亲和力，适合年轻化社交、个人理财、习惯追踪等面向 Z 世代用户的移动应用。
