# Corporate Blue Light — Style Guide

tags: light-mode, corporate, clean, blue-accent, webapp, shadows, rounded

## Description
专业企业级 SaaS 视觉方案，以蓝色为主调传递可信赖感。浅色背景配合清晰的层级关系和微妙阴影，适合后台管理、CRM、项目管理等需要长时间使用的工具型产品。

## Color System
| Token | Value | Usage |
|-------|-------|-------|
| bg-page | #F8FAFC | 页面背景 |
| bg-card | #FFFFFF | 卡片/容器背景 |
| bg-surface | #F1F5F9 | 抬升表面（hover、active 状态底） |
| bg-accent | #2563EB | 强调色背景（按钮、选中态） |
| text-primary | #0F172A | 主文字 |
| text-secondary | #475569 | 次要文字 |
| text-muted | #94A3B8 | 弱化文字（占位符、禁用态） |
| text-on-accent | #FFFFFF | 强调背景上的文字 |
| border | #E2E8F0 | 边框、分割线 |
| accent | #2563EB | 主强调色 |
| accent-hover | #1D4ED8 | 强调色 hover |
| success | #16A34A | 成功 |
| error | #DC2626 | 错误 |
| warning | #D97706 | 警告 |
