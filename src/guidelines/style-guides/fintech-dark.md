# Fintech Dark — Style Guide

## Description
金融科技暗色界面，深灰底色配合蓝紫色强调营造精密专业感。紧凑间距和双字体系统（UI 用无衬线、数据用等宽）适合数据密集的交易面板、资产管理、风控仪表盘等金融场景。
