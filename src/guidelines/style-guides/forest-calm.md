# Forest Calm — Style Guide

## Description
深林绿与奶油白的沉静组合，灵感来自北欧森林。低饱和绿色系配合宽松间距营造缓慢呼吸的节奏感。适合冥想应用、自然教育、园艺工具、慢生活社区等与自然连接的产品。
