# Glassmorphism Violet — Style Guide

tags: dark-mode, vibrant, purple-accent, rounded, gradient, mobile, soft, webapp

## Description
紫色系毛玻璃风格，深色基底上叠加半透明层次感。色彩饱和且通透，适合现代移动端应用、音乐播放器、社交产品等追求视觉冲击力的年轻化产品。
