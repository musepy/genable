# Material Ocean — Style Guide

## Description
深海蓝底色配合青色强调的 Material Design 变体。分层阴影系统建立清晰的 Z 轴层级，圆角统一为 8px。适合工具型 SaaS、项目管理、云服务控制台等需要信息密度和视觉清晰度并重的产品。
