# Midnight Gold — Style Guide

tags: dark-mode, luxury, elegant, gold-accent, minimal, sharp, monochrome, webapp

## Description
纯黑基底配合金色强调的奢华方案。克制的装饰、精确的间距和低调的金属感传递高端质感。适合奢侈品电商、高端会员体系、精品酒店预订等追求 premium 感的产品。
