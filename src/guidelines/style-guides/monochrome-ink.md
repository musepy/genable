# Monochrome Ink — Style Guide

## Description
纯灰阶无彩色方案，用描边代替填充、用层次代替色彩来建立视觉节奏。连成功/错误状态都用灰度表达，极致的去色让内容（尤其是图片和视频）成为唯一色彩来源。适合摄影作品集、设计师 portfolio、画廊官网。
