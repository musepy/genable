# Neon Cyber — Style Guide

## Description
赛博朋克霓虹风格，纯黑底上用品红和青色制造电子化的视觉张力。高饱和度撞色配合几何造型和微弱发光阴影，适合游戏面板、电竞社区、音乐可视化、科幻主题产品。
