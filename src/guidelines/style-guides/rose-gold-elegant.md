# Rose Gold Elegant — Style Guide

tags: light-mode, elegant, warm, soft, rounded, dual-tone, luxury, webapp

## Description
玫瑰金与深灰的优雅搭配，用低饱和暖粉色调传递精致感而不显甜腻。温润的金属质感配合适度圆角和细腻的阴影，适合美妆电商、女性社区、婚礼策划、精品咖啡等追求格调的产品。
