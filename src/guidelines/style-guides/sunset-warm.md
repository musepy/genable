# Sunset Warm — Style Guide

tags: light-mode, warm, vibrant, orange-accent, rounded, soft, playful, landing-page

## Description
日落橙红色系，在暖白底上铺开活力与热情。柔和圆角和温暖色调避免攻击性，用色彩梯度引导视觉流向。适合社交平台、美食应用、旅行预订、活动推广等需要情绪感染力的产品。
