# Swiss Grid — Style Guide

## Description
瑞士国际主义设计风格，严格网格、零装饰、红色强调。用极端的排版对比（大标题 + 小正文）和负空间建立视觉张力。适合建筑事务所、设计工作室、博物馆官网等追求理性美学的场景。
