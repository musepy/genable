# Terminal Dark — Style Guide

tags: dark-mode, developer, monospace, sharp-corners, green-accent, minimal, crisp

## Description
面向开发者的暗色工具界面，灵感来自终端和 IDE。零圆角、monospace 字体、绿色强调色营造精确的技术感。适合 CLI 工具、API 文档、代码编辑器、DevOps 面板等开发者产品。

## Color System
| Token | Value | Usage |
|-------|-------|-------|
| bg-page | #0D1117 | 页面背景 |
| bg-card | #161B22 | 卡片/容器背景 |
| bg-surface | #21262D | 抬升表面（hover、active 状态底） |
| bg-accent | #238636 | 强调色背景（按钮、选中态） |
| text-primary | #E6EDF3 | 主文字 |
| text-secondary | #8B949E | 次要文字 |
| text-muted | #484F58 | 弱化文字（占位符、禁用态） |
| text-on-accent | #FFFFFF | 强调背景上的文字 |
| border | #30363D | 边框、分割线 |
| accent | #238636 | 主强调色 |

| success | #238636 | 成功 |
| error | #DA3633 | 错误 |
