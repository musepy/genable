# /src/knowledge

> 组件知识库 - 为 LLM 提供结构化的 UI 组件设计规则。
> 包含组件结构、变体、语义颜色、最佳实践 (Do/Don't)。
> 被 `contextBuilder.ts` 调用，注入到 System Prompt 中。

---

## 文件清单 (File Inventory)

| 文件 | 地位 | 功能 |
|------|------|------|
| `index.ts` | 入口 | 重导出所有知识模块 |
| `componentKnowledge.ts` | **核心** | Badge/Button 组件知识定义 |
| `intentRecognizer.ts` | **核心** | 用户意图识别 (prompt → intent) |

---

## 扩展指南

添加新组件知识：
1. 在 `componentKnowledge.ts` 创建 `{Component}Knowledge` 接口
2. 定义 structure, variants, semanticColors, dos, donts
3. 添加到 `getComponentKnowledge()` 函数
4. 在 `intentRecognizer.ts` 的 `INTENT_PATTERNS` 添加关键词
