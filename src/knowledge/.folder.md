# /src/knowledge

> 组件知识库 - 为 LLM 提供结构化的 UI 组件设计规则。
> 包含项目组件注册表、设计令牌、以及 UI 渲染模板。

---

## 知识源定位

| 来源 | 用途 | 注入方式 | 特点 |
| :--- | :--- | :--- | :--- |
| `.agent/knowledge/` | **结构蓝图 (Anatomy)** | 工具调用 | 仅包含节点层次结构与语义，无数值约束，赋予 LLM 设计自由度 |
| `src/knowledge/components/` | **技术规范 (Spec)** | 工具调用 | 包含组件 Props 契约与 Code Snippets (Few-shot)，用于精准实现 |
| `src/generated/` | **预编译知识包** | 构建时嵌入 | 包含样式、颜色、排版等基础设计原子 |
| `src/schema/` | **DSL 验证 (Schema)** | 后处理器 | 负责验证 LLM 输出的 NodeLayer 是否符合运行时规范 |
